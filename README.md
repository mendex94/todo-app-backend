# ToDo App Fastify API

---

This is a simple ToDo App API built with Fastify and MongoDB as the database and Redis for caching. The API applies clean architecture principles and is built with TypeScript.

## Libraries

- [Fastify](https://www.fastify.io/)
- [Prisma](https://www.prisma.io/)
- [MongoDB](https://www.mongodb.com/)
- [Redis](https://redis.io/)
- [Vitest](https://vitejs.dev/)
- [TypeScript](https://www.typescriptlang.org/)
- [Zod](https://github.com/colinhacks/zod)
